Sistema de Tarefas (To-Do List) 

Este é um sistema CRUD completo de gerenciamento de tarefas desenvolvido em PHP com banco de dados MySQL, utilizando a biblioteca PDO (Prepared Statements) para maior segurança e Bootstrap para a interface.

Tecnologias Utilizadas
  PHP 8+ (Lógica, Sessões e Autenticação)
  MySQL (Banco de Dados)
  HTML5 & CSS3 + **Bootstrap 5** (Interface do Usuário)

## ⚙️ Como executar o projeto

1. Tenha um servidor local instalado (como XAMPP, WAMP ou Laragon).
2. Clone este repositório ou baixe os arquivos e coloque a pasta do projeto dentro do diretório do seu servidor (ex: `htdocs` no XAMPP).
3. Abra o seu gerenciador de banco de dados (ex: MySQlWorckbench) e rode o script SQL que está no arquivo `txt_SQL.txt` para criar o banco de dados `tarefas` e as tabelas necessárias.
4. Acesse o projeto pelo navegador: `http://localhost/tarefas/login.php` (ajuste o caminho se o nome da sua pasta for diferente).
